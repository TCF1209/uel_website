# UEL — Unique Excellent Lubricant

Brochure website for **UEL (Unique Excellent Lubricant)**, a Malaysian engine oil and lubricants brand.

> Premium engine oils and lubricants engineered for Malaysian roads. Three drops, six families, one standard.

## Stack

- Pure HTML / CSS / JS (no build step)
- Static `index.html` and `products.html`
- Deployed as a static site on Vercel

## Local preview

```bash
# any static file server works — pick one
python3 -m http.server 5173
# or
npx serve .
```

Then open http://localhost:5173

## Deploy

```bash
vercel deploy --prod
```

1. Passenger Car Engine Oil (PCMO)
2. Motorcycle Oil 4T / 2T (MCO)
3. Heavy Duty Diesel (HDDO)
4. Gear & Transmission (ATF / CVTF / Gear Oils)
5. Industrial Fluids (Hydraulic / Compressor)
6. Auto Fluids & Additives (Brake / Coolant / Power Steering)
